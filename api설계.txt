메모 프로젝트 api 설계서
공통 url : http://localhost:8080 (모든 url은 이 url로 시작함)


-게시글 실행 컨트롤러 
PostController
@RequestMapping("/api/board") 설정되어있음 

- 추가 과제 Spring lv3.  2023/06/20 이후 
1. 회원가입 
Method : Post
url : /api/auth/signup
request : username(사용자명), password(비밀번호), isAdmin(관리자여부)
response : DB에 중복된 username이 없다면 회원을 저장하고 Client 로 성공했다는 메시지, 상태코드 반환

2. 로그인 
Method : Post
url : /api/auth/login
request : username(사용자명), password(비밀번호)
response : 로그인 성공 시, 로그인에 성공한 유저의 정보와 JWT를 활용하여 토큰을 발급하고, 
발급한 토큰을 Header에 추가하고 성공했다는 메시지, 상태코드 와 함께 Client에 반환

3. 게시글 작성
Method : Post
url : /api/board/post
request : title, content
response : 저장된 게시글을 Client 로 반환
Request Header : Authorization: Bearer "토큰값" 

4. 게시글 목록 조회 
Method : Get
url : /api/board/posts
request :
response : 게시글 반환
Request Header : Authorization: Bearer "토큰값" 

5. 게시글 상세 조회 
Method : Get
url : /api/board/post/{id}
request : 글id 
response : 글id에 해당하는 게시글 반환
Request Header : Authorization: Bearer "토큰값" 

6. 게시글 수정
Method : Put
url : /api/board/post/{id}
request : title, content, password, token
response : 저장된 게시글을 Client 로 반환
Request Header : Authorization: Bearer "토큰값" 

7. 게시글 삭제 
Method : Delete 
url : /api/board/post/{id}
request : id, password, token
response : 삭제 메시지 호출
Request Header : Authorization: Bearer "토큰값"

8. 댓글 등록 
Method : Post 
url : /api/board/post/{id}/comment
request : id, token, HttpServletRequest, CommentRequestDto
response : 등록된 데이터 호출
Request Header : Authorization: Bearer "토큰값"

9. 댓글 수정 
Method : Put 
url : /api/board/post/comment/{id}
request : id, token, HttpServletRequest, CommentRequestDto
response : 수정된 데이터 호출
Request Header : Authorization: Bearer "토큰값"

10. 댓글 삭제 
Method : Delete 
url : /api/board/post/comment/{id}
request : id, token, HttpServletRequest, CommentRequestDto
response : 삭제 메시지 호출
Request Header : Authorization: Bearer "토큰값"
