1. 전체조회 
Method : get
url : /api/board/postlist
request : 
response : List<Post>(게시글 객체 목록)
2. 상세조회
Method : get
url : /api/board/post/1
request : 게시글번호
response : Post(게시글 객체)
3. 등록
Method : Post
url : /api/board/post
request : Post(게시글 객체)
response : Post(게시글 객체) 
4. 수정
Method : Put
url : /api/board/post/update/2
request : Post(게시글 객체), 게시글번호
response : Post(게시글 객체)
5. 삭제 
Method : Delete
url : /api/board/post/delete/2
request : Post(게시글 객체), 게시글번호
response : 비밀번호가 맞으면 삭제후 삭제되었습니다. 맞지않으면 삭제하지않고 
비밀번호가 맞지않는다 표시 



- 추가 과제 Spring lv2.  2023/06/20 이후 
6. 회원가입 
Method : Post
url : /api/auth/signup
request : username(사용자명), password(비밀번호), isAdmin(관리자여부)
response : DB에 중복된 username이 없다면 회원을 저장하고 Client 로 성공했다는 메시지, 상태코드 반환
7. 로그인 
Method : Post
url : /api/auth/login
request : username(사용자명), password(비밀번호)
response : 로그인 성공 시, 로그인에 성공한 유저의 정보와 JWT를 활용하여 토큰을 발급하고, 
발급한 토큰을 Header에 추가하고 성공했다는 메시지, 상태코드 와 함께 Client에 반환
